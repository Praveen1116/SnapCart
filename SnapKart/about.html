The Mission


SnapCart is a sleek, front-end-only e-commerce project built to showcase elegant design, seamless cart functionality, and the magic of JavaScript. It’s not just a cart—it’s a learning journey in pixels and logic.

The Maker Story


Created by a passionate developer exploring the power of HTML, CSS, and JavaScript. This project is part of a larger vision to build immersive, user-centric applications while sharpening front-end development skills.

Features

  • Dark Mode
  • Add/Remove to Cart
  • Increase/Decrease items in Cart

Tech Stack

  • JavaScript
